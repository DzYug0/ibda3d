# Ibda3D - 3D Printing E-Commerce Platform

Ibda3D is a premier destination for 3D printing supplies in Algeria. This project is a modern e-commerce application built with React, Supabase, and Tailwind CSS.

## Features

-   **Product Catalog**: Browse 3D printers, filaments, and accessories.
-   **User Accounts**: Secure login, profile management, and order history.
-   **Admin Dashboard**: Manage products, orders, users, and view analytics.
-   **Cart & Checkout**: Seamless shopping experience with coupon support.
-   **Reviews**: User-submitted product reviews and ratings.
-   **Multi-language**: Support for English and Arabic (RTL).

## Tech Stack

-   **Frontend**: React, Vite, TypeScript
-   **Styling**: Tailwind CSS, Shadcn UI
-   **Backend**: Supabase (PostgreSQL, Auth, Storage, Edge Functions)
-   **State Management**: React Query, Context API

## Getting Started

1.  Clone the repository.
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Start the development server:
    ```bash
    npm run dev
    ```

## Deployment

This project is configured for deployment on Vercel. Ensure you set up the necessary environment variables for Supabase connectivity.
